# 🧩 Business Case Study Analysis 

## 📌 Reducing Cart Abandonment in E-commerce

---

## 📖 Project Overview

This project analyzes customer behavior in an e-commerce platform to identify reasons for cart abandonment and propose data-driven solutions to improve conversion rates.

The analysis is based on a dataset of **1,000 user sessions**, tracking user journey from product browsing to purchase completion.

---

## 🎯 Objectives

* Reduce cart abandonment rate by ~20%
* Improve checkout experience
* Increase overall conversion rate
* Identify key drop-off points in user journey

---

## 📊 Dataset Description

The dataset includes:

* User sessions
* Device type (Mobile/Desktop)
* Product categories
* Cart, checkout, and purchase activity
* Abandonment reasons

### 4️⃣ Abandonment Reasons

* Payment-related drop-offs
* High shipping cost
* Slow website performance
* Low-intent browsing

👉 Payment friction and cost transparency are key contributors

---

## 🧠 Root Cause Analysis

Using the 5 Whys technique, the main issue identified is:
👉 Inefficient checkout system with limited payment options

---

## 🔄 Process Improvement

**AS-IS:**
User → Cart → Checkout → Payment → Drop

**TO-BE:**
User → Cart → Cost Visibility → Fast Checkout → Multiple Payment Options → Purchase

---

## ⚙️ Requirements

### Functional Requirements

* Guest checkout
* Multiple payment methods (UPI, cards, wallets)
* Cart save feature
* Upfront cost display

### Non-Functional Requirements

* Page load time < 3 seconds
* Secure payment gateway
* Mobile-first responsive design

---

## 💡 Proposed Solutions

* Improve payment system and add multiple gateways
* Reduce checkout steps
* Show total cost upfront
* Optimize mobile experience

---

## 📊 KPIs & Success Metrics

* Reduce cart abandonment (70.7% → ~56%)
* Increase conversion rate (20% → ~28%)
* Improve checkout completion rate

---

## 📌 Conclusion

The analysis indicates that the checkout stage is the primary drop-off point, mainly due to payment friction and hidden costs.

Improving checkout flow, payment options, and mobile UX can significantly enhance conversion rates and reduce revenue loss.

---

## ⚠️ Assumptions & Limitations

* Dataset is simulated
* User intent is inferred from behavior
* External factors (pricing strategy, competition) not included
